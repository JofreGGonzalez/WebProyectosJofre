Software de gestión de facturación y presupuestos

Objetivo▼

El objetivo de este proyecto es crear desde cero una aplicación de escritorio para mi madre para crear y gestionar tanto la facturación mensual como los presupuestos, clientes, contratos, gestorías y recibos. Pudiendo así no solo crear y administrar todos los puntos mencionados, sino también imprimir la información mediante plantillas.

Dando contexto al motivo por el cual se ha creado y realizado este proyecto, es debido a que en el pasado mi tío le creó una aplicación mediante Visual Basic a mi madre para que pudiese hacer las gestiones de su empresa. Actualmente, la tecnología ha avanzado mucho y nuevos requerimientos han surgido, al igual que hay funcionalidades que ya no necesita. La idea es crear una aplicación a medida para las necesidades de mi madre manteniendo la estética del programa antiguo.

La aplicación se ha hecho usando el lenguaje de programación Java y una base de datos Access.

Motivación▼

  • Poner en práctica mis conocimientos con el lenguaje de programación Java

  • Poner en práctica en un proyecto real y ampliar mis conocimientos de diseño y consultas de bases de datos.

Modelo de datos▼

Originalmente, en el programa antiguo se usó una base de datos Access con tablas sin ninguna relación entre ellas, es decir, eran islas de datos y no tenían relación alguna entre tablas.

Para el nuevo programa se decidió cambiar esta estructura de la base de datos y usar un modelo de datos relacional. Adicionalmente, se ha replanteado todo el modelo.
